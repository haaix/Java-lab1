# 阅读程序

## 实验目的
1.基本掌握本人所选择的集成开发工具的使用方法。<br/>
2.掌握Java源程序命名、运行方法。<br/>
3.理解、掌握创建包的方法。<br/>
4.初步了解类的实例化方法。

## 实验过程
1.阅读程序的源文件，理解其功能、含义，分析三个Java源程序文件的关系，理解三个源文件表达的实体业务逻辑关系（银行账户、存款人员、测试类）。<br/>
2.在开发环境中组织代码并运行。<br/>
3.在Customer类新添加一个属性，描述其年龄，并补充操作年龄的方法。在测试类中调用这些方法。<br/>
4.在理解上述程序的情况下，尝试从头分析并复写该程序。
